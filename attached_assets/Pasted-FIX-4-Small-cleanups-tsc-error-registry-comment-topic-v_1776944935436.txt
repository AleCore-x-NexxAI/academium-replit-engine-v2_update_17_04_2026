D. Fix the nested-Tooltip collision in the cross-discipline picker
In client/src/components/CanonicalCaseCreator.tsx at the framework row rendering (around lines 707-748), the current structure is:

When selectedElsewhere is true, wrap ONLY the <span> with the framework name in the outer tooltip, not the whole div.
Leave the HelpCircle with its own description tooltip untouched.
