# E1. Extend DecisionEvidenceLogEntry Schema

**Problem:** The schema at `shared/schema.ts:428` is missing three fields that Part D AI prompts and dashboard endpoints depend on:
- `student_input` — needed so AI prompts can reference what the student actually wrote
- `classification` — PASS / NUDGE / BLOCK, needed for turn annotations (Spec 3.3)
- `evidence_quotes` — per-competency short quotes from student input, needed by `class-patterns` endpoint which reads `log.evidence_quotes` at `routes.ts:3402` (currently always undefined)

**Fix:** Extend the interface in `shared/schema.ts` around line 428:

```typescript
export interface DecisionEvidenceLogEntry {
  // EXISTING FIELDS — keep unchanged:
  signals_detected: SignalExtractionEntry;
  rds_score: number | null;
  rds_band: "SURFACE" | "ENGAGED" | "INTEGRATED" | null;
  competency_evidence: CompetencyEvidenceEntry;
  raw_signal_scores: {
    intent: number;
    justification: number;
    tradeoffAwareness: number;
    stakeholderAwareness: number;
    ethicalAwareness: number;
  };
  isMcq?: boolean;

  // NEW FIELDS (required for dashboard analysis):
  student_input?: string;
  classification?: "PASS" | "NUDGE" | "BLOCK";
  evidence_quotes?: Partial<Record<"C1" | "C2" | "C3" | "C4" | "C5", string>>;
  turn_number?: number;
}
```

**Compatibility:** All new fields are optional with `?` so existing session state (before this deploy) continues to load without migration. Only newly-processed turns will populate them.

**No Drizzle migration needed** — these fields live inside the JSON `currentState` column, not as separate columns.

# E2. Write the New Fields in director.ts at Every Turn

**Problem:** `director.ts:1026` constructs the `evidenceEntry` for the PASS path but doesn't include the new fields.

**Fix:** Extend the `evidenceEntry` construction in `server/agents/director.ts` around line 1026. The `competency_evidence` object already contains the per-competency levels — now derive a short quote per competency from `evidenceLog.signals_detected`:

```typescript
const existingEvidenceLogs = context.decisionEvidenceLogs || [];

// Derive evidence_quotes from the strongest signal per competency
const competencyQuoteMap: Record<string, string | undefined> = {};
// Competency → primary signal mapping per Engine Packet v2.0:
// C1 (analytical) ← justification
// C2 (strategic)  ← intent
// C3 (stakeholder)← stakeholderAwareness
// C4 (ethical)    ← ethicalAwareness
// C5 (tradeoff)   ← tradeoffAwareness
const compToSignal: Record<string, keyof typeof evidenceLog.signals_detected> = {
  C1: "justification",
  C2: "intent",
  C3: "stakeholderAwareness",
  C4: "ethicalAwareness",
  C5: "tradeoffAwareness",
};
for (const [comp, sigKey] of Object.entries(compToSignal)) {
  const sig = evidenceLog.signals_detected[sigKey];
  if (sig && sig.extracted_text && sig.quality >= 1) {
    competencyQuoteMap[comp] = sig.extracted_text.substring(0, 160);
  }
}

const evidenceEntry: import("@shared/schema").DecisionEvidenceLogEntry = {
  // EXISTING FIELDS:
  signals_detected: {
    intent: { quality: evidenceLog.signals_detected.intent.quality as 0 | 1 | 2 | 3, extracted_text: evidenceLog.signals_detected.intent.extracted_text },
    justification: { quality: evidenceLog.signals_detected.justification.quality as 0 | 1 | 2 | 3, extracted_text: evidenceLog.signals_detected.justification.extracted_text },
    tradeoffAwareness: { quality: evidenceLog.signals_detected.tradeoffAwareness.quality as 0 | 1 | 2 | 3, extracted_text: evidenceLog.signals_detected.tradeoffAwareness.extracted_text },
    stakeholderAwareness: { quality: evidenceLog.signals_detected.stakeholderAwareness.quality as 0 | 1 | 2 | 3, extracted_text: evidenceLog.signals_detected.stakeholderAwareness.extracted_text },
    ethicalAwareness: { quality: evidenceLog.signals_detected.ethicalAwareness.quality as 0 | 1 | 2 | 3, extracted_text: evidenceLog.signals_detected.ethicalAwareness.extracted_text },
  },
  rds_score: evidenceLog.rds_score,
  rds_band: evidenceLog.rds_band as "SURFACE" | "ENGAGED" | "INTEGRATED" | null,
  competency_evidence: evidenceLog.competency_evidence,
  raw_signal_scores: evidenceLog.raw_signal_scores,
  isMcq: evidenceLog.isMcq,

  // NEW FIELDS:
  student_input: context.studentInput,
  classification: "PASS",
  evidence_quotes: competencyQuoteMap,
  turn_number: currentDecisionNum,
};
```

**NUDGE / BLOCK paths** — these do NOT append to `decisionEvidenceLogs` (they pass through `context.decisionEvidenceLogs` unchanged), because NUDGE asks for re-attempt and BLOCK rejects. This is correct behavior — the evidence log is only appended when a decision is actually committed via PASS. **Do not change this.**

**Do not break the existing flow.** All new fields are optional. Existing tests should continue to pass.

# E4. Backfill Helper for Existing Completed Sessions

**Problem:** Alejandro's test session (and any other already-completed sessions) have empty `decisionEvidenceLogs` and `framework_detections` because they ran before Part B was deployed. Without a backfill, the meeting demo will be an empty dashboard.

**Fix:** Add a one-shot admin endpoint that walks a scenario's completed sessions and reconstructs the missing arrays from the `turns` table (which HAS the raw `studentInput` + `agentResponse` per turn).

**New endpoint** in `server/routes.ts` (placed near the other dashboard endpoints, around line 3040, admin-only):

```typescript
app.post("/api/admin/scenarios/:scenarioId/backfill-analysis", isAuthenticated, async (req: any, res) => {
  try {
    const userId = req.user.claims.sub;
    const user = await storage.getUser(userId);
    if (!user || user.role !== "admin") {
      return res.status(403).json({ message: "Admin access required" });
    }

    const { scenarioId } = req.params;
    const scenario = await storage.getScenario(scenarioId);
    if (!scenario) return res.status(404).json({ message: "Scenario not found" });

    const initialState = scenario.initialState as any;
    const language = initialState?.language || scenario.language || "es";
    const frameworks = initialState?.frameworks || [];

    const sessionsData = await getSessionsWithTurns(scenarioId);
    const completed = sessionsData.filter(s => s.session.status === "completed");

    const { extractSignals } = await import("./agents/signalExtractor");
    const { detectFrameworks } = await import("./agents/frameworkDetector");
    const { computeCompetencyEvidence } = await import("./agents/director");
    // ^^ If computeCompetencyEvidence isn't exported, inline the logic or export it.

    let processedSessions = 0, processedTurns = 0, errors = 0;

    for (const s of completed) {
      const existingState = (s.session.currentState as any) || {};
      const existingLogs: any[] = existingState.decisionEvidenceLogs || [];
      const existingFwDets: any[][] = existingState.framework_detections || [];

      // Only backfill if logs are actually missing/empty
      if (existingLogs.length >= s.turns.length && existingFwDets.length >= s.turns.length) continue;

      const newLogs: any[] = [];
      const newFwDets: any[][] = [];

      for (const turn of s.turns) {
        try {
          // Reconstruct the context enough to extract signals
          const agentResp: any = turn.agentResponse || {};
          const updatedState = agentResp.updatedState || {};
          const isMcq = (turn as any).format === "multiple_choice";

          let signalsDetected, rdsScore, rdsBand, rawScores, competencyEvidence;

          // If the agentResponse already contains evidenceLog data (Part B turns), use it directly
          if (updatedState?.decisionEvidenceLogs?.length) {
            const lastLog = updatedState.decisionEvidenceLogs[updatedState.decisionEvidenceLogs.length - 1];
            signalsDetected = lastLog.signals_detected;
            rdsScore = lastLog.rds_score;
            rdsBand = lastLog.rds_band;
            rawScores = lastLog.raw_signal_scores;
            competencyEvidence = lastLog.competency_evidence;
          } else {
            // Old turn — re-extract signals from stored studentInput
            const extractCtx = {
              studentInput: turn.studentInput,
              language,
              scenario: { ...scenario, initialState },
              sessionId: s.session.id,
              currentDecision: turn.turnNumber,
              // minimal context
            } as any;
            const extracted = await extractSignals(extractCtx);
            signalsDetected = extracted.signals_detected;
            rdsScore = extracted.rds_score;
            rdsBand = extracted.rds_band;
            rawScores = extracted.raw_signal_scores;
            competencyEvidence = extracted.competency_evidence;
          }

          // Evidence quotes — derive from signals
          const compToSignal: Record<string, string> = {
            C1: "justification",
            C2: "intent",
            C3: "stakeholderAwareness",
            C4: "ethicalAwareness",
            C5: "tradeoffAwareness",
          };
          const evidenceQuotes: Record<string, string> = {};
          for (const [comp, sigKey] of Object.entries(compToSignal)) {
            const sig = (signalsDetected as any)[sigKey];
            if (sig?.extracted_text && sig.quality >= 1) {
              evidenceQuotes[comp] = sig.extracted_text.substring(0, 160);
            }
          }

          newLogs.push({
            signals_detected: signalsDetected,
            rds_score: rdsScore,
            rds_band: rdsBand,
            competency_evidence: competencyEvidence,
            raw_signal_scores: rawScores,
            isMcq,
            student_input: turn.studentInput,
            classification: "PASS",
            evidence_quotes: evidenceQuotes,
            turn_number: turn.turnNumber,
          });

          // Reconstruct framework detections
          if (!isMcq && frameworks.length > 0) {
            const fwDets = detectFrameworks(turn.studentInput, signalsDetected, frameworks, language);
            newFwDets.push(fwDets);
          } else {
            newFwDets.push([]);
          }

          processedTurns++;
        } catch (err) {
          console.error(`[Backfill] Failed turn ${turn.turnNumber} session ${s.session.id}:`, err);
          errors++;
          newLogs.push(null);
          newFwDets.push([]);
        }
      }

      // Write back to session
      const mergedState = {
        ...existingState,
        decisionEvidenceLogs: newLogs.filter(l => l !== null),
        framework_detections: newFwDets,
      };

      await storage.updateSimulationSession(s.session.id, { currentState: mergedState });
      processedSessions++;
    }

    // Invalidate dashboard cache for this scenario
    const cacheKeys = ["class-stats", "module-health", "depth-trajectory", "class-patterns"];
    for (const k of cacheKeys) dashboardCache.delete(`${k}-${scenarioId}`);

    res.json({ processedSessions, processedTurns, errors, totalCompleted: completed.length });
  } catch (error) {
    console.error("[Backfill] Fatal error:", error);
    res.status(500).json({ message: "Backfill failed", error: String(error) });
  }
});
```

**Requirements for the helper:**
- Admin-only — enforce via role check.
- Idempotent — skip sessions whose logs are already fully populated.
- Non-destructive — only adds missing logs, never overwrites existing ones.
- Invalidates dashboard cache for that scenario so the next dashboard load reflects the backfill.
- Returns counts of processed sessions / turns / errors for visibility.

**How to run it:** Santi hits `POST /api/admin/scenarios/<scenarioId>/backfill-analysis` once per scenario with broken data. No UI needed — this is an admin one-shot.

**Guarantee:** If `extractSignals` cannot be called with a minimal context (e.g., it requires full pipeline state), the helper should instead just use whatever is already in `turn.agentResponse.updatedState.decisionEvidenceLogs` (newer turns) and reconstruct only what's missing. Do not regenerate signals if doing so would cost significant tokens — prefer preserving whatever was captured originally.
